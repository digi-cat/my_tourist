Мой турист

Бесплатный открытый аналитический сервис на открытых данных для продвижения туристических продуктов региона РФ.

Сервис «Мой турист» предложен, как решение задачи Анализ цифрового следа туриста на конкурсе Цифровых Решений для регионов (сейчас World AI&Data Сhallenge) и стал победителем в 2019 году.

Продукт позволяет провести оценку объема потенциальной целевой аудитории и автоматически разбить ее на группы (сегменты), в рамках которых потребители имеют схожие или аналогичные запросы. Сегментация произведена по местоположению, полу, возрастным группам, семейному положению, наличию детей, интересам и доходу. Целевые группы могут использоваться для дальнейшей проработки аватаров клиентов под конкретное место отдыха или мероприятие. А затем, для создания и настройки рекламы для онлайн- и оффлайн-продвижения.

Сервис «Мой турист» может помочь как продвижению существующих услуг, так и выводу на рынок новых, исходя из местонахождения, объема и интересов потенциальной аудитории, вывести туристические объекты региона на федеральный уровень.

Карта цифрового присутствия объектов туризма позволяет определить уровень осведомленности и заинтересованности потенциальной аудитории, а следовательно, помогает создавать содержание рекламных сообщений согласно модели иерархии воздействия.

Источники данных

Тепловая карта цифрового присутствия

Тепловая карта цифрового присутствия курортов в регионах построена при помощи сервиса Яндекс.Подбор слов. Расчеты основаны на региональной популярности ключевых фраз, связанных с важнейшими курортами и достопримечательностями. Региональная популярность (affinity index) — это доля, которую занимает регион в показах по данному слову, деленная на долю всех показов результатов поиска, пришедшихся на этот регион. Тепловая карта курортных мест может быть построена для любого региона. Список ключевых фраз задается в разделе Настройки.

Целевая аудитория

Целевые группы потенциальных клиентов определяются на основе данных социальной сети ВКонтакте при помощи API ВКонтакте. Географические и демографические признаки берутся из анкетных данных пользователей сети. Данные по интересам автоматически собираются и анализируются на стороне ВКонтакте на основе взаимодействия пользователей с контентом и рекламой сети, посещением тематических сообществ и внешних сайтов.

Распределения уровня заработной платы

Распределения уровня заработной платы по регионам получены из данных МТС. Отсутствующие данные заменяются усредненными значениями для половозрастной группы.

Численность населения

Источником данных по численности населения регионов является Росстат.
